This application will let you search through the songs available in a playlist on Spotify; specifically, "The Longest Mixtape: 1000 Songs for You" by Caribou.

If you're interested, you can read more about him here, from Merge Records!

Available route paths

  • /(nothing)

    - main index-grid

  • /(your name)

    - main index that says hello!

  • /check/song/(song title)

    - will see if a song with this name is available

  • /check/arist/(artist name)

    - will check if this artist has any songs in this playlist
Please note these routes are all case-sensitive! Please be sure you are using correct capitalization and puncutation in the URLs.

Try these suggested routes:

  • /(put your name here)
  • /check/song/Ghost
  • /check/artist/Moodymann
